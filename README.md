# bilibili-m7panel-FunctionalityEnhancementPlugin
# Bilibili高级弹幕增强

这是一个专门用于增强B站高级弹幕功能的脚本，通过优化原有高级弹幕面板并添加新功能，让高级弹幕的发送更加便捷和丰富。

✨ 主要功能

1️⃣ 界面优化
★ 颜色选择器优化：添加系统原生取色器，让颜色选择更加直观
★ 字体扩展：新增「楷体」「幼圆」「华文彩云」实用字体选项
★ 精确时间控制：支持毫秒级精度的弹幕发送时间
★ 强大的坐标计算功能

2️⃣ 样式弹幕增强
★ 立体阴影效果
- 自定义阴影颜色
- 可调整X/Y轴偏移
- 打造立体感十足的弹幕

★ 颜色描边效果
- 自定义描边颜色
- 灵活调整描边间距
- 让弹幕更加醒目清晰

★ 文字背景效果
- 自定义背景颜色和字符
- 支持文字竖排显示
- 创造独特的装饰效果

🔧 使用方法
1. 打开任意B站视频
2. 点击高级弹幕按钮
3. 即可使用增强功能
